Welcome to the home page of MPTCE

What is this?

MPTCE is a Windows application that aims to enable people to use surface electromyography (sEMG) data acquisition devices at home for biofeedback training. This kind of training can help amputee patients on learning how to make the most effective use of bionic hand prostheses.

How does it work?

MPTCE translates the intended hand movements executed by the user into control events that can be sent to third party software such as games. Movement detection is achieved by using pattern recognition algorithms that analyze multiple channels of realtime sEMG data. These data must be typically obtained through some sEMG data acquisition device. Getting MPTCE to recognise a set of hand movements involves the following steps:

1- Record a sample of the movements to be recognised: out-of-the-box available hand movements are open, close, flexion, extension, pronation and supination, as well as possible combinations of two or three of these six basic movements.

2- Run a feature extraction process: the recording is divided into 200ms-long windows. Each window is processed to extract a set of selectable features that characterise the sEMG signal at that point in time.

3- Train a pattern recognition algorithm: the features extracted on step 2 are presented to a user-selectable pattern recognition algorithm through a learning process.

4- Use the trained pattern recognition algorithm to detect movements in real time: currently, only support for simulating keyboard events is included. Each movement can be mapped to any of the available keyboard events.

Additionally, MPTCE allows to perform movement detection based solely on the amplitude of the acquired sEMG signal reaching an user-defined level (threshold-based activation). This method can yield good results when only a reduced number of channels (1 or 2) is available. No feature extraction or training is required in this case, but a separate sample recording process as for pattern recognition based techniques must be carried out.
