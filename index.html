Welcome to the home page of MPTCE

What is this?

MPTCE is a Windows application that aims to enable people to use surface electromyography (sEMG) data acquisition devices at home for biofeedback training. This can help amputee patients during the process of learning how to make the most effective use of bionic hand prostheses.

How does it work?

MPTCE translates muscle activations associated with hand movements executed by the user into control events. These control events can then be sent to third-party software such as games. Movement detection is achieved by using pattern recognition algorithms that analyze multiple channels of real-time sEMG data. These data are obtained through an sEMG data acquisition device. Getting MPTCE to recognise a set of hand movements involves the following steps:

  1. Recording a sample of the movements to be recognised: Available hand movements are open, close, flexion, extension, pronation and supination, as well as possible combinations of two or three of these six basic movements.

  2. Running a feature extraction process: the recording is divided into small partially overlapping temporal windows. Each window is processed to extract a set of selectable features that characterise the sEMG signal at that point in time.

  3. Training a pattern recognition algorithm: the features extracted on step 2 are presented to a user-selectable pattern recognition algorithm through a learning process.

  4. Detecting movements in real time: currently, only keyboard events are supported. Each movement can be mapped to any of the available keyboard events.

Additionally, MPTCE supports movement detection based on the amplitude of the acquired sEMG signal reaching an user-defined level (threshold-based activation). This simpler method can yield good results when only a reduced number of channels (e.g. less than 3) are available. No feature extraction or training is required when using threshold-based activation, but a separate sample recording process from that used for pattern recognition must be carried out.
